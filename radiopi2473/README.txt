Radio Pi README:

To use the RadioPi App, use this folder hierarchy: in the default user pi
/home/pi/piremote and run you server there
Or simply make sure you have the folder radiopi2473 in the folder where you run the server

Source for the Pi2
https://github.com/SandPox/fm_transmitter/
compile with: make -j4

Original source:
http://www.icrobotics.co.uk/wiki/index.php/Turning_the_Raspberry_Pi_Into_an_FM_Transmitter
BEWARE: All infos given are correct, except the pinout used in the linked youtube demo by a Matt Richardson is wrong.
GPIO 7, is pin 4, just above the GPIO label on your board
